#### 介绍
SSM栈的，皮鱼后端
已把之前的piyu-backend项目集成到此项目，作为一个模块
此项目将有两个maven项目，一是皮鱼api，二是皮于后台

发现不改成maven多模块就无法引入其它项目的类，所以还是改成了多模块
其实很多东西是遇到时才能想到解决办法

其实我更喜欢beta版，但这个项目我想要稳重点

#### 软件架构
软件架构说明
